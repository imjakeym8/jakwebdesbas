My Emmet Cheat Sheet!

The HTML form indicated here tells HTML:

"Get information from this form after a user types his first name, and last name from the text fields."

The HTML form attributes here include:

"action"
- defines a location where the form is specified for,
"target"
- defines in what form displays the response after submitting the form; and,
"method"
- this is the HTTP method the browser uses to submit the form.

for this particular HTML form, it uses:

"#"
- which the form response is specified for the same url where it is submitted;
"_self"
- which loads the response in the same HTML fram as the current one; and,
"GET"
- which data is created to the "action" attribute URI with a '?' as a separator, and URI is sent to the server.
